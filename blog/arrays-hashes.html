Arrays Vs Hashes (Ruby)

What is an Array

Arrays are collections of any objects which are associated with integers corresponding with their order in the array.

What is a Hash?

A hash is a collection of key-value pairs in which a key has a specific value given to it by a user.

What is the difference between Arrays and Hashes

The main difference between Arrays and Hashes, is that objects in an array are classified strictly with integers, while objects in a Hash can be given any value.

Examples

Array:

my_number_array = [1,2,3,4,5,6]

Hash:

kids_favorite_vegetables = { "Donny" => "Carrot", "Jake" => "Broccoli" "Sarah" => "Potato"}
